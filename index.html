    Cephei is a framework for jailbroken iOS devices that includes various convenience features for developers. Primarily, it focuses on settings-related features, but it also contains other utilties. I hope you’ll appreciate what it has to offer.

  • Opener

    Opener is a developer framework for overriding link destinations. Handler packages instruct Opener whether to change a URL that’s about to be opened, or is being interacted with by the user. For example, Maps links could be redirected to the Google Maps app.

    TypeStatus Plus includes a framework for developers to create their own providers – plugins that listen for and display notifications through TypeStatus’ subtle notification interface.
